## MasterKat 
MasterKat is a fictional premium chocolate brand created exclusively for this academic project. The main goal of this project was to develop practical experience in Front-End Development, applying essential concepts of HTML and CSS while creating a modern and visually attractive landing page.

## About the Project

This website was designed to represent a premium chocolate company, highlighting elegance, quality, and sophistication through its visual identity.

During the development process, several front-end techniques were practiced to improve structure, design, and user experience.

## Technologies Used
HTML5
CSS3
Boxicons
Google Fonts

## Features Implemented
Responsive page structure
Styled navigation header
Sections for About, Ingredients, and Branches
Custom Google Fonts integration
Boxicons for visual elements
Hover effects and transitions
Pseudo-classes for interaction styling
Pseudo-elements for decorative details
Organized layout using containers and spacing techniques

## Fonts Used
Playfair Display (titles)
Poppins (navigation/menu)
Lato (text content)

## Learning Goals
This project was created to gain hands-on experience with:

Semantic HTML structure
CSS styling best practices
Visual hierarchy and typography
Navigation menu creation
Pseudo-classes (:hover, :focus, :nth-child)
Pseudo-elements (::before, ::after, ::first-letter)
Basic UI/UX concepts

## Purpose
More than just a fictional chocolate brand, MasterKat represents a learning journey in front-end development and the process of transforming ideas into a real interface.
